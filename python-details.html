Python- simplified

Python is a popular computer programming language often used to create websites and software, automate tasks, and conduct data analysis. Python is a versatile language that can be used for a variety of different purposes.

  • Last Update : August 23, 2019
  • Category : Web Design

Description

This course provides an introduction to Python language. It will teach you how to create basic programs, and how to use the Python language to solve problems. Students will learn about core programming concepts like data structures, conditionals, loops, and variables.This course includes an overview of the various tools available to write and run Python, gets students to code quickly, and provides hands-on projects on coding with commonly used data structures, writing custom functions, and...and write to files. This course may be more powerful than some of the other introductory Python courses, as it delves deeper into some of the core programming topics.

What You will Learn?

  • Describe the fundamental concepts of the Python language.
  • Recognize and use fundamental programming concepts like variables, loops, conditionals, and data structures
  • Create fully-functional Python scripts by utilizing custom functions, widely-used data structures, and file reading and writing.
  • Utilize several tools to create and execute Python code.
